# Thruline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `thruline` — a Rust CLI and DSL runtime for defining and executing deterministic multiagent pipelines with self-contained runner definitions, typed artifact flow, conditional routing, and parallel fan-out/fan-in.

**Architecture:** Three-layer stack: parser (pest grammar → AST) → validator (static checks) → runtime (state machine + artifact store + scheduler). Execution is driven by a pluggable `Driver` trait. The stdio driver emits `stage_invoke` events, checkpoints state to `~/.thruline/runs/<id>/state.json`, and exits — allowing the harness to invoke agents and call `thruline resume`. A companion Claude Code skill reads NDJSON events and drives the pipeline from within an agent harness.

**Tech Stack:** Rust (2021 edition), pest 2 (PEG parser), clap 4 (CLI), serde/serde_json (events + state), tokio (async runtime), thiserror (typed errors), anyhow (CLI error propagation), uuid v4 (run IDs), chrono (timestamps), dirs 5 (home dir), reqwest 0.12 (API driver)

### Task 3: Pest Grammar

**Files:**
- Create: `src/parser/grammar.pest`

**Grammar design notes:**
- `model:` accepts an unquoted identifier (e.g. `claude-opus-4-8`) or quoted string
- `in:` / `out:` accept one or more space-separated `artifact_decl` entries (no comma needed — the `as` keyword unambiguously separates them from the next field)
- Route fan-out: `target[*N?]` with `parallel` keyword; fan-in: `source[*]` without `parallel`

- [ ] **Step 1: Write grammar.pest**

```pest
// src/parser/grammar.pest

WHITESPACE = _{ " " | "\t" | "\n" | "\r" }
COMMENT    = _{ "//" ~ (!"\n" ~ ANY)* }

// Primitives
identifier  = @{ ASCII_ALPHA ~ (ASCII_ALPHANUMERIC | "_" | "-")* }
model_id    = @{ ASCII_ALPHA ~ (ASCII_ALPHANUMERIC | "_" | "-" | ".")* }
quoted_str  = @{ "\"" ~ (!"\"" ~ ANY)* ~ "\"" }
pos_int     = @{ ASCII_NONZERO_DIGIT ~ ASCII_DIGIT* }
float_val   = @{ ASCII_DIGIT+ ~ ("." ~ ASCII_DIGIT+)? }
string_val  =  { quoted_str | model_id }

// Prompt/file sources
file_ref    =  { "file" ~ "(" ~ quoted_str ~ ")" }
prompt_val  =  { file_ref | quoted_str }

// Artifact declarations
artifact_kind = { "file" | "ref" }
seed_init     = { "(" ~ quoted_str ~ ")" }
artifact_decl = { identifier ~ "?"? ~ "as" ~ artifact_kind ~ seed_init? }

// ── Runner ────────────────────────────────────────────────────────────────
tools_list   = { "[" ~ (identifier ~ ("," ~ identifier)*)? ~ "]" }
runner_model = { "model" ~ ":" ~ string_val }
runner_sys   = { "system" ~ ":" ~ prompt_val }
runner_tools = { "tools" ~ ":" ~ tools_list }
runner_temp  = { "temperature" ~ ":" ~ float_val }
runner_maxt  = { "max_tokens" ~ ":" ~ pos_int }
runner_field = { runner_model | runner_sys | runner_tools | runner_temp | runner_maxt }
runner_decl  = { "runner" ~ identifier ~ "{" ~ runner_field* ~ "}" }

// ── Stage ─────────────────────────────────────────────────────────────────
stage_in     = { "in" ~ ":" ~ artifact_decl+ }
stage_out    = { "out" ~ ":" ~ artifact_decl+ }
stage_agent  = { "agent" ~ ":" ~ identifier }
stage_prompt = { "prompt" ~ ":" ~ prompt_val }
stage_format = { "format" ~ ":" ~ identifier }
stage_field  = { stage_in | stage_out | stage_agent | stage_prompt | stage_format }
stage_decl   = { "stage" ~ identifier ~ "{" ~ stage_field* ~ "}" }

// ── Routes ────────────────────────────────────────────────────────────────
cmp_op       = { "==" | "!=" }
predicate    = { identifier ~ "." ~ identifier ~ cmp_op ~ quoted_str }
fan_in_src   = { identifier ~ "[*]" }
stage_src    = { identifier }
route_source = { predicate | fan_in_src | stage_src }

fan_out_spec = { "[*" ~ pos_int? ~ "]" }
stage_target = { identifier ~ fan_out_spec? }
route        = { route_source ~ "->" ~ stage_target ~ "parallel"? }

// ── Pipeline ──────────────────────────────────────────────────────────────
pipe_start   = { "start" ~ ":" ~ identifier }
routes_block = { "routes" ~ "{" ~ route* ~ "}" }
pipeline_decl = { "pipeline" ~ identifier ~ "{" ~ pipe_start ~ routes_block ~ "}" }

// ── Import ────────────────────────────────────────────────────────────────
import_decl  = { "import" ~ quoted_str }

// ── Top-level ─────────────────────────────────────────────────────────────
tl_item = _{ import_decl | runner_decl | stage_decl | pipeline_decl }
tl_file =  { SOI ~ tl_item* ~ EOI }
```

// src/parser/mod.rs
use pest::iterators::Pair;
use pest::Parser;
use pest_derive::Parser;
use std::path::Path;
use crate::ast::*;

#[derive(Parser)]
#[grammar = "parser/grammar.pest"]
pub struct TlParser;

pub fn parse_file(path: &Path) -> anyhow::Result<Vec<TlItem>> {
    let src = std::fs::read_to_string(path)?;
    parse_str(&src)
}

pub fn parse_str(src: &str) -> anyhow::Result<Vec<TlItem>> {
    let file = TlParser::parse(Rule::tl_file, src)
        .map_err(|e| anyhow::anyhow!("parse error: {}", e))?
        .next()
        .unwrap();

    let mut items = Vec::new();
    for pair in file.into_inner() {
        match pair.as_rule() {
            Rule::import_decl  => items.push(parse_import(pair)),
            Rule::runner_decl  => items.push(TlItem::Runner(parse_runner(pair))),
            Rule::stage_decl   => items.push(TlItem::Stage(parse_stage(pair))),
            Rule::pipeline_decl => items.push(TlItem::Pipeline(parse_pipeline(pair))),
            Rule::EOI          => {}
            _ => {}
        }
    }
    Ok(items)
}

fn parse_import(pair: Pair<Rule>) -> TlItem {
    let path = unquote(pair.into_inner().next().unwrap().as_str());
    TlItem::Import(path)
}

fn parse_runner(pair: Pair<Rule>) -> RunnerDecl {
    let mut inner = pair.into_inner();
    let name = inner.next().unwrap().as_str().to_string();
    let mut model = String::new();
    let mut system = PromptSource::Inline(String::new());
    let mut tools = Vec::new();
    let mut temperature = None;
    let mut max_tokens = None;

    for field in inner {
        let mut fi = field.into_inner();
        match fi.next().unwrap().as_rule() {
            Rule::runner_model => {
                model = parse_string_val(fi.next().unwrap());
            }
            Rule::runner_sys => {
                system = parse_prompt_val(fi.next().unwrap());
            }
            Rule::runner_tools => {
                tools = fi.next().unwrap().into_inner()
                    .map(|t| t.as_str().to_string())
                    .collect();
            }
            Rule::runner_temp => {
                temperature = Some(fi.next().unwrap().as_str().parse().unwrap());
            }
            Rule::runner_maxt => {
                max_tokens = Some(fi.next().unwrap().as_str().parse().unwrap());
            }
            _ => {}
        }
    }
    RunnerDecl { name, model, system, tools, temperature, max_tokens }
}

fn parse_stage(pair: Pair<Rule>) -> StageDecl {
    let mut inner = pair.into_inner();
    let name = inner.next().unwrap().as_str().to_string();
    let mut inputs = Vec::new();
    let mut outputs = Vec::new();
    let mut agent = String::new();
    let mut prompt = None;
    let mut format = None;

    for field in inner {
        let mut fi = field.into_inner();
        match fi.next().unwrap().as_rule() {
            Rule::stage_in => {
                inputs = fi.map(parse_artifact_decl).collect();
            }
            Rule::stage_out => {
                outputs = fi.map(parse_artifact_decl).collect();
            }
            Rule::stage_agent => {
                agent = fi.next().unwrap().as_str().to_string();
            }
            Rule::stage_prompt => {
                prompt = Some(parse_prompt_val(fi.next().unwrap()));
            }
            Rule::stage_format => {
                format = Some(fi.next().unwrap().as_str().to_string());
            }
            _ => {}
        }
    }
    StageDecl { name, inputs, outputs, agent, prompt, format }
}

fn parse_artifact_decl(pair: Pair<Rule>) -> ArtifactDecl {
    let mut inner = pair.into_inner();
    let name_tok = inner.next().unwrap().as_str();
    let optional = name_tok.ends_with('?');
    let name = name_tok.trim_end_matches('?').to_string();

    // pest: optional marker is separate token
    let mut next = inner.next().unwrap();
    let optional_flag = next.as_rule() == Rule::artifact_kind && false; // placeholder
    // The grammar has `identifier ~ "?"? ~ "as" ~ artifact_kind ~ seed_init?`
    // pest captures the literal "?" as a separate token if present
    // Re-parse: check if next is "?" or artifact_kind directly
    let kind_str;
    let mut seed_path = None;

    // After identifier, next might be the literal "?" or artifact_kind
    if next.as_str() == "?" {
        // optional marker consumed, advance
        next = inner.next().unwrap(); // "as" is silent in WHITESPACE, so next is artifact_kind
        kind_str = next.as_str();
    } else if next.as_str() == "as" {
        next = inner.next().unwrap();
        kind_str = next.as_str();
    } else {
        kind_str = next.as_str();
    }

    let kind = match kind_str {
        "file" => ArtifactKind::File,
        "ref"  => ArtifactKind::Ref,
        _      => ArtifactKind::Ref,
    };

    if let Some(seed) = inner.next() {
        if seed.as_rule() == Rule::seed_init {
            let s = seed.into_inner().next().unwrap().as_str();
            seed_path = Some(unquote(s));
        }
    }

    ArtifactDecl { name, optional, kind, seed_path }
}

fn parse_pipeline(pair: Pair<Rule>) -> PipelineDecl {
    let mut inner = pair.into_inner();
    let name = inner.next().unwrap().as_str().to_string();
    let start = inner.next().unwrap().into_inner().next().unwrap().as_str().to_string();
    let routes_pair = inner.next().unwrap();
    let routes = routes_pair.into_inner().map(parse_route).collect();
    PipelineDecl { name, start, routes }
}

fn parse_route(pair: Pair<Rule>) -> Route {
    let mut inner = pair.into_inner();
    let source_pair = inner.next().unwrap();
    let target_pair = inner.next().unwrap();
    let parallel = inner.next().map(|p| p.as_str() == "parallel").unwrap_or(false);

    let source = match source_pair.as_rule() {
        Rule::predicate => {
            let mut pi = source_pair.into_inner();
            let stage = pi.next().unwrap().as_str().to_string();
            let artifact = pi.next().unwrap().as_str().to_string();
            let op = match pi.next().unwrap().as_str() {
                "==" => CompareOp::Eq,
                _    => CompareOp::Ne,
            };
            let value = unquote(pi.next().unwrap().as_str());
            RouteSource::Predicate { stage, artifact, op, value }
        }
        Rule::fan_in_src => {
            RouteSource::FanIn(source_pair.into_inner().next().unwrap().as_str().to_string())
        }
        Rule::stage_src => {
            RouteSource::Stage(source_pair.into_inner().next().unwrap().as_str().to_string())
        }
        _ => RouteSource::Stage(source_pair.as_str().to_string()),
    };

    let mut ti = target_pair.into_inner();
    let stage = ti.next().unwrap().as_str().to_string();
    let parallel_spec = ti.next().map(|spec| {
        let limit = spec.into_inner().next()
            .map(|n| n.as_str().parse::<u32>().unwrap());
        ParallelSpec { limit }
    });

    Route { source, target: RouteTarget { stage, parallel_spec }, parallel }
}

fn parse_prompt_val(pair: Pair<Rule>) -> PromptSource {
    let inner = pair.into_inner().next().unwrap();
    match inner.as_rule() {
        Rule::file_ref => {
            let s = inner.into_inner().next().unwrap().as_str();
            PromptSource::File(unquote(s))
        }
        _ => PromptSource::Inline(unquote(inner.as_str())),
    }
}

fn parse_string_val(pair: Pair<Rule>) -> String {
    let inner = pair.into_inner().next().unwrap();
    match inner.as_rule() {
        Rule::quoted_str => unquote(inner.as_str()),
        _                => inner.as_str().to_string(),
    }
}

fn unquote(s: &str) -> String {
    s.trim_matches('"').to_string()
}

```rust
// src/validator.rs
use std::collections::{HashMap, HashSet};
use thiserror::Error;
use crate::ast::*;

#[derive(Debug, Error, PartialEq)]
pub enum ValidationError {
    #[error("stage '{stage}' references unknown runner '{runner}'")]
    UnknownRunner { stage: String, runner: String },
    #[error("route references unknown stage '{0}'")]
    UnknownStage(String),
    #[error("route predicate references unknown artifact '{stage}.{artifact}'")]
    UnknownArtifact { stage: String, artifact: String },
    #[error("parallel fan-out '[*]' for stage '{0}' has no matching fan-in route")]
    UnpairedFanOut(String),
    #[error("concurrency limit must be >= 1 in fan-out for stage '{0}'")]
    InvalidConcurrencyLimit(String),
    #[error("circular import: {0}")]
    CircularImport(String),
    #[error("duplicate name '{0}'")]
    DuplicateName(String),
    #[error("runner '{0}' is missing required field 'model'")]
    MissingModel(String),
    #[error("runner '{0}' is missing required field 'system'")]
    MissingSystem(String),
}

#[derive(Debug)]
pub struct Warning(pub String);

pub struct ValidationResult {
    pub errors: Vec<ValidationError>,
    pub warnings: Vec<Warning>,
}

pub fn validate(items: &[TlItem]) -> ValidationResult {
    let mut errors = Vec::new();
    let mut warnings = Vec::new();

    // Build index maps
    let mut runner_names: HashSet<String> = HashSet::new();
    let mut stage_names: HashSet<String>  = HashSet::new();
    let mut pipeline_names: HashSet<String> = HashSet::new();
    let mut stage_outputs: HashMap<String, HashSet<String>> = HashMap::new();

    // Check for duplicates and build indexes
    for item in items {
        match item {
            TlItem::Runner(r) => {
                if !runner_names.insert(r.name.clone()) {
                    errors.push(ValidationError::DuplicateName(r.name.clone()));
                }
                if r.model.is_empty() {
                    errors.push(ValidationError::MissingModel(r.name.clone()));
                }
                if matches!(&r.system, PromptSource::Inline(s) if s.is_empty()) {
                    errors.push(ValidationError::MissingSystem(r.name.clone()));
                }
            }
            TlItem::Stage(s) => {
                if !stage_names.insert(s.name.clone()) {
                    errors.push(ValidationError::DuplicateName(s.name.clone()));
                }
                let outs: HashSet<String> = s.outputs.iter().map(|a| a.name.clone()).collect();
                stage_outputs.insert(s.name.clone(), outs);
            }
            TlItem::Pipeline(p) => {
                if !pipeline_names.insert(p.name.clone()) {
                    errors.push(ValidationError::DuplicateName(p.name.clone()));
                }
            }
            TlItem::Import(_) => {}
        }
    }

    // Validate stage agent refs and cross-check runners
    for item in items {
        if let TlItem::Stage(s) = item {
            if !runner_names.contains(&s.agent) {
                errors.push(ValidationError::UnknownRunner {
                    stage: s.name.clone(),
                    runner: s.agent.clone(),
                });
            }
        }
    }

    // Validate pipeline routes
    for item in items {
        let TlItem::Pipeline(p) = item else { continue };

        // Collect fan-out targets and fan-in sources for pairing check
        let mut fan_out_stages: HashSet<String> = HashSet::new();
        let mut fan_in_stages:  HashSet<String> = HashSet::new();

        if !stage_names.contains(&p.start) {
            errors.push(ValidationError::UnknownStage(p.start.clone()));
        }

        // Collect stages referenced in routes for unreachable warning
        let mut referenced_stages: HashSet<String> = HashSet::new();

        for route in &p.routes {
            match &route.source {
                RouteSource::Stage(s) | RouteSource::FanIn(s) => {
                    if !stage_names.contains(s) {
                        errors.push(ValidationError::UnknownStage(s.clone()));
                    }
                    referenced_stages.insert(s.clone());
                    if matches!(&route.source, RouteSource::FanIn(_)) {
                        fan_in_stages.insert(s.clone());
                    }
                }
                RouteSource::Predicate { stage, artifact, .. } => {
                    if !stage_names.contains(stage) {
                        errors.push(ValidationError::UnknownStage(stage.clone()));
                    } else if let Some(outs) = stage_outputs.get(stage) {
                        if !outs.contains(artifact) {
                            errors.push(ValidationError::UnknownArtifact {
                                stage: stage.clone(),
                                artifact: artifact.clone(),
                            });
                        }
                    }
                    referenced_stages.insert(stage.clone());
                }
            }

            let target = &route.target.stage;
            if !stage_names.contains(target) {
                errors.push(ValidationError::UnknownStage(target.clone()));
            }
            referenced_stages.insert(target.clone());

            if let Some(spec) = &route.target.parallel_spec {
                if spec.limit == Some(0) {
                    errors.push(ValidationError::InvalidConcurrencyLimit(target.clone()));
                }
                fan_out_stages.insert(target.clone());
            }
        }

        // Every fan-out must have a corresponding fan-in
        for fo_stage in &fan_out_stages {
            if !fan_in_stages.contains(fo_stage) {
                errors.push(ValidationError::UnpairedFanOut(fo_stage.clone()));
            }
        }

        // Warn on unreachable stages
        for name in &stage_names {
            if name != &p.start && !referenced_stages.contains(name) {
                warnings.push(Warning(format!("stage '{}' is unreachable in pipeline '{}'", name, p.name)));
            }
        }
    }

    ValidationResult { errors, warnings }
}
```

use std::collections::HashMap;
use std::path::{Path, PathBuf};
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct ArtifactStore {
    pub files: HashMap<String, PathBuf>,
    pub refs:  HashMap<String, String>,
}

impl ArtifactStore {
    pub fn new() -> Self {
        Self::default()
    }

    pub fn set_file(&mut self, key: impl Into<String>, path: impl Into<PathBuf>) {
        self.files.insert(key.into(), path.into());
    }

    pub fn set_ref(&mut self, key: impl Into<String>, value: impl Into<String>) {
        self.refs.insert(key.into(), value.into());
    }

    pub fn get_file(&self, key: &str) -> Option<&PathBuf> {
        self.files.get(key)
    }

    pub fn get_ref(&self, key: &str) -> Option<&str> {
        self.refs.get(key).map(|s| s.as_str())
    }

    pub fn file_keys(&self) -> impl Iterator<Item = &String> {
        self.files.keys()
    }

    pub fn ref_keys(&self) -> impl Iterator<Item = &String> {
        self.refs.keys()
    }

    /// Snapshot all artifacts as a JSON object for event payloads
    pub fn to_json(&self) -> serde_json::Value {
        let mut map = serde_json::Map::new();
        for (k, v) in &self.files {
            map.insert(k.clone(), serde_json::Value::String(v.to_string_lossy().into_owned()));
        }
        for (k, v) in &self.refs {
            map.insert(k.clone(), serde_json::Value::String(v.clone()));
        }
        serde_json::Value::Object(map)
    }
}

### Task 8: Run State

Run state is the serializable snapshot of a pipeline run. It is saved to `~/.thruline/runs/<run-id>/state.json` after each `stage_invoke` so that `resume` can load it and continue.

- [ ] **Step 1: Write runtime/state.rs**

```rust
// src/runtime/state.rs
use chrono::{DateTime, Utc};
use serde::{Deserialize, Serialize};
use std::path::PathBuf;
use crate::runtime::artifact::ArtifactStore;

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub enum RunStatus {
    Running,
    AwaitingResume { stage: String },
    ParallelAwait  { stage: String, remaining: usize },
    Done,
    Failed(String),
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RunState {
    pub run_id:    String,
    pub pipeline:  String,
    pub tl_file:   PathBuf,
    pub status:    RunStatus,
    pub artifacts: ArtifactStore,
    pub history:   Vec<String>,
    pub started:   DateTime<Utc>,
    pub updated:   DateTime<Utc>,
}

impl RunState {
    pub fn new(run_id: String, pipeline: String, tl_file: PathBuf) -> Self {
        let now = Utc::now();
        Self {
            run_id,
            pipeline,
            tl_file,
            status: RunStatus::Running,
            artifacts: ArtifactStore::new(),
            history: Vec::new(),
            started: now,
            updated: now,
        }
    }

    pub fn run_dir(&self) -> PathBuf {
        runs_dir().join(&self.run_id)
    }

    pub fn state_path(&self) -> PathBuf {
        self.run_dir().join("state.json")
    }

    pub fn events_path(&self) -> PathBuf {
        self.run_dir().join("events.ndjson")
    }

    pub fn artifacts_dir(&self) -> PathBuf {
        self.run_dir().join("artifacts")
    }

    pub fn save(&mut self) -> anyhow::Result<()> {
        self.updated = Utc::now();
        let dir = self.run_dir();
        std::fs::create_dir_all(&dir)?;
        std::fs::create_dir_all(self.artifacts_dir())?;
        let json = serde_json::to_string_pretty(self)?;
        std::fs::write(self.state_path(), json)?;
        Ok(())
    }

    pub fn load(run_id: &str) -> anyhow::Result<Self> {
        let path = runs_dir().join(run_id).join("state.json");
        let json = std::fs::read_to_string(&path)
            .map_err(|_| anyhow::anyhow!("run '{}' not found", run_id))?;
        Ok(serde_json::from_str(&json)?)
    }
}

pub fn runs_dir() -> PathBuf {
    dirs::home_dir()
        .expect("cannot determine home directory")
        .join(".thruline")
        .join("runs")
}

pub fn list_runs() -> anyhow::Result<Vec<RunState>> {
    let dir = runs_dir();
    if !dir.exists() {
        return Ok(vec![]);
    }
    let mut runs = Vec::new();
    for entry in std::fs::read_dir(&dir)? {
        let entry = entry?;
        let state_path = entry.path().join("state.json");
        if state_path.exists() {
            if let Ok(json) = std::fs::read_to_string(&state_path) {
                if let Ok(state) = serde_json::from_str::<RunState>(&json) {
                    runs.push(state);
                }
            }
        }
    }
    runs.sort_by(|a, b| b.started.cmp(&a.started));
    Ok(runs)
}

### Task 9: Scheduler

**Files:**
- Create: `src/runtime/scheduler.rs`

The scheduler manages parallel fan-out: it holds the list of work items, the concurrency limit, and tracks which are running vs. pending. For the stdio driver, parallel slots are processed one invocation at a time (each triggers a resume cycle). For the API driver, all slots up to the limit can be running concurrently.

- [ ] **Step 1: Write runtime/scheduler.rs**

```rust
// src/runtime/scheduler.rs
use std::collections::VecDeque;
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ParallelJob {
    pub slot: usize,
    pub input: serde_json::Value,
    pub output: Option<serde_json::Value>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Scheduler {
    pub stage: String,
    pub limit: Option<u32>,
    pub jobs: Vec<ParallelJob>,
    pub next_slot: usize,
    pub completed: usize,
}

impl Scheduler {
    pub fn new(stage: String, items: Vec<serde_json::Value>, limit: Option<u32>) -> Self {
        let jobs = items.into_iter().enumerate().map(|(i, input)| ParallelJob {
            slot: i,
            input,
            output: None,
        }).collect();
        Self { stage, limit, jobs, next_slot: 0, completed: 0 }
    }

    pub fn total(&self) -> usize {
        self.jobs.len()
    }

    pub fn is_done(&self) -> bool {
        self.completed == self.jobs.len()
    }

    /// Returns the next slot index that should be dispatched, respecting the
    /// concurrency limit (number of in-flight jobs = next_slot - completed).
    pub fn next_to_dispatch(&self) -> Option<usize> {
        let in_flight = self.next_slot - self.completed;
        let limit = self.limit.map(|l| l as usize).unwrap_or(usize::MAX);
        if self.next_slot < self.jobs.len() && in_flight < limit {
            Some(self.next_slot)
        } else {
            None
        }
    }

    pub fn dispatch(&mut self) -> Option<&ParallelJob> {
        if let Some(idx) = self.next_to_dispatch() {
            self.next_slot += 1;
            Some(&self.jobs[idx])
        } else {
            None
        }
    }

    pub fn complete_slot(&mut self, slot: usize, output: serde_json::Value) {
        if let Some(job) = self.jobs.iter_mut().find(|j| j.slot == slot) {
            job.output = Some(output);
            self.completed += 1;
        }
    }

    pub fn collect_results(&self) -> Vec<serde_json::Value> {
        let mut results: Vec<_> = self.jobs.iter()
            .filter_map(|j| j.output.clone())
            .collect();
        results
    }
}

### Task 10: Runtime State Machine

**Files:**
- Modify: `src/runtime/mod.rs`

The runtime evaluates routes given the current artifact store, dispatches `invoke_agent` calls through the driver, and advances the state machine. For the stdio driver, each `advance()` call emits one `stage_invoke` event and returns — the process then exits; `resume` restarts with the stored state.

- [ ] **Step 1: Write route evaluation logic in runtime/mod.rs**

```rust
// src/runtime/mod.rs
pub mod artifact;
pub mod scheduler;
pub mod state;

use crate::ast::*;
use crate::driver::{AgentInvocation, Driver};
use crate::events::{RunnerSpec, ThrulineEvent};
use crate::runtime::artifact::ArtifactStore;
use crate::runtime::state::{RunState, RunStatus};
use std::collections::HashMap;
use std::path::Path;

pub struct Runtime {
    pub state: RunState,
    pub items: Vec<TlItem>,
}

impl Runtime {
    pub fn new(state: RunState, items: Vec<TlItem>) -> Self {
        Self { state, items }
    }

    fn runners(&self) -> HashMap<String, &RunnerDecl> {
        self.items.iter().filter_map(|i| {
            if let TlItem::Runner(r) = i { Some((r.name.clone(), r)) } else { None }
        }).collect()
    }

    fn stages(&self) -> HashMap<String, &StageDecl> {
        self.items.iter().filter_map(|i| {
            if let TlItem::Stage(s) = i { Some((s.name.clone(), s)) } else { None }
        }).collect()
    }

    fn pipeline(&self, name: &str) -> Option<&PipelineDecl> {
        self.items.iter().find_map(|i| {
            if let TlItem::Pipeline(p) = i { if p.name == name { return Some(p); } }
            None
        })
    }

    /// Evaluate routes for the given source stage and return the matching target.
    pub fn evaluate_routes(
        &self,
        from_stage: &str,
        routes: &[Route],
        artifacts: &ArtifactStore,
    ) -> Option<(&Route, String)> {
        for route in routes {
            let matches = match &route.source {
                RouteSource::Stage(s) => s == from_stage,
                RouteSource::FanIn(s) => s == from_stage,
                RouteSource::Predicate { stage, artifact, op, value } => {
                    if stage != from_stage { continue; }
                    let key = format!("{}.{}", stage, artifact);
                    let actual = artifacts.get_ref(&key).unwrap_or("");
                    match op {
                        CompareOp::Eq => actual == value,
                        CompareOp::Ne => actual != value,
                    }
                }
            };
            if matches {
                return Some((route, route.target.stage.clone()));
            }
        }
        None
    }

    /// Resolve a RunnerDecl to a RunnerSpec, loading file-based system prompts.
    pub fn resolve_runner(&self, runner: &RunnerDecl, base_path: &Path) -> anyhow::Result<RunnerSpec> {
        let system = match &runner.system {
            PromptSource::Inline(s) => s.clone(),
            PromptSource::File(rel) => {
                let abs = base_path.parent().unwrap_or(Path::new(".")).join(rel);
                std::fs::read_to_string(&abs)
                    .map_err(|_| anyhow::anyhow!("system prompt file not found: {}", abs.display()))?
            }
        };
        Ok(RunnerSpec {
            name: runner.name.clone(),
            model: runner.model.clone(),
            system,
            tools: runner.tools.clone(),
            temperature: runner.temperature,
            max_tokens: runner.max_tokens,
        })
    }

    /// Build the artifact map for a stage's inputs from the store.
    fn stage_input_artifacts(&self, stage: &StageDecl, artifacts: &ArtifactStore) -> serde_json::Value {
        let mut map = serde_json::Map::new();
        for input in &stage.inputs {
            let key = format!("{}.{}", stage.name, input.name);
            if let Some(path) = artifacts.get_file(&key) {
                map.insert(input.name.clone(), serde_json::json!(path.to_string_lossy()));
            } else if let Some(val) = artifacts.get_ref(&key) {
                map.insert(input.name.clone(), serde_json::json!(val));
            }
        }
        serde_json::Value::Object(map)
    }

    /// Advance the pipeline one step: invoke the next stage and save state.
    pub async fn advance(&mut self, driver: &dyn Driver) -> anyhow::Result<bool> {
        let pipeline_name = self.state.pipeline.clone();
        let pipeline = self.pipeline(&pipeline_name)
            .ok_or_else(|| anyhow::anyhow!("pipeline '{}' not found", pipeline_name))?;

        let current_stage_name = match &self.state.status {
            RunStatus::Running => pipeline.start.clone(),
            RunStatus::AwaitingResume { stage } => stage.clone(),
            RunStatus::Done | RunStatus::Failed(_) => return Ok(false),
            RunStatus::ParallelAwait { .. } => return Ok(false),
        };

        let stages = self.stages();
        let stage = stages.get(&current_stage_name)
            .ok_or_else(|| anyhow::anyhow!("stage '{}' not found", current_stage_name))?;

        let runners = self.runners();
        let runner_decl = runners.get(&stage.agent)
            .ok_or_else(|| anyhow::anyhow!("runner '{}' not found", stage.agent))?;

        let tl_path = self.state.tl_file.clone();
        let runner_spec = self.resolve_runner(runner_decl, &tl_path)?;

        let input_artifacts = self.stage_input_artifacts(stage, &self.state.artifacts);
        let prompt = match &stage.prompt {
            Some(PromptSource::Inline(s)) => Some(s.clone()),
            Some(PromptSource::File(p)) => {
                let abs = tl_path.parent().unwrap_or(Path::new(".")).join(p);
                Some(std::fs::read_to_string(&abs)?)
            }
            None => None,
        };

        let invocation = AgentInvocation {
            run_id: self.state.run_id.clone(),
            stage: current_stage_name.clone(),
            runner: runner_spec.clone(),
            artifacts: input_artifacts.clone(),
            prompt: prompt.clone(),
        };

        ThrulineEvent::StageInvoke {
            run_id: self.state.run_id.clone(),
            ts: chrono::Utc::now(),
            stage: current_stage_name.clone(),
            runner: runner_spec,
            artifacts: input_artifacts,
            prompt,
        }.emit();

        self.state.status = RunStatus::AwaitingResume { stage: current_stage_name };
        self.state.save()?;

        driver.invoke_agent(invocation).await?;
        Ok(true)
    }

    /// Feed the result of a completed stage back into the state machine.
    pub fn resume_stage(
        &mut self,
        stage_name: &str,
        outputs: Vec<(String, String, bool)>, // (artifact_name, value, is_file)
    ) -> anyhow::Result<()> {
        for (name, value, is_file) in outputs {
            let key = format!("{}.{}", stage_name, name);
            if is_file {
                self.state.artifacts.set_file(&key, &value);
            } else {
                self.state.artifacts.set_ref(&key, &value);
            }
        }

        self.state.history.push(stage_name.to_string());

        let pipeline_name = self.state.pipeline.clone();
        let pipeline = self.pipeline(&pipeline_name)
            .ok_or_else(|| anyhow::anyhow!("pipeline not found"))?;

        if let Some((route, next_stage)) = self.evaluate_routes(
            stage_name,
            &pipeline.routes,
            &self.state.artifacts,
        ) {
            ThrulineEvent::RouteTaken {
                run_id: self.state.run_id.clone(),
                ts: chrono::Utc::now(),
                from: stage_name.to_string(),
                to: next_stage.clone(),
                predicate: format!("{:?}", route.source),
            }.emit();

            self.state.status = RunStatus::AwaitingResume { stage: next_stage };
        } else {
            // No matching route — pipeline done
            let outputs: Vec<String> = self.state.artifacts.file_keys().cloned().collect();
            ThrulineEvent::PipelineDone {
                run_id: self.state.run_id.clone(),
                ts: chrono::Utc::now(),
                outputs,
            }.emit();
            self.state.status = RunStatus::Done;
        }

        self.state.save()?;
        Ok(())
    }
}

use async_trait::async_trait;
use reqwest::header::{HeaderMap, HeaderValue, CONTENT_TYPE};
use serde::{Deserialize, Serialize};
use crate::driver::{AgentInvocation, AgentResult, Driver};

pub struct ApiDriver {
    pub api_key: String,
}

impl ApiDriver {
    pub fn new(api_key: String) -> Self {
        Self { api_key }
    }

    pub fn from_env() -> anyhow::Result<Self> {
        let key = std::env::var("ANTHROPIC_API_KEY")
            .map_err(|_| anyhow::anyhow!("ANTHROPIC_API_KEY environment variable not set"))?;
        Ok(Self::new(key))
    }
}

#[derive(Serialize)]
struct MessagesRequest {
    model: String,
    max_tokens: u32,
    system: String,
    messages: Vec<Message>,
    #[serde(skip_serializing_if = "Option::is_none")]
    temperature: Option<f64>,
}

#[derive(Serialize, Deserialize)]
struct Message {
    role: String,
    content: String,
}

#[derive(Deserialize)]
struct MessagesResponse {
    content: Vec<ContentBlock>,
}

#[derive(Deserialize)]
struct ContentBlock {
    #[serde(rename = "type")]
    kind: String,
    text: Option<String>,
}

#[async_trait]
impl Driver for ApiDriver {
    async fn invoke_agent(&self, invocation: AgentInvocation) -> anyhow::Result<AgentResult> {
        let user_content = build_user_message(&invocation);

        let req = MessagesRequest {
            model: invocation.runner.model.clone(),
            max_tokens: invocation.runner.max_tokens.unwrap_or(8096),
            system: invocation.runner.system.clone(),
            messages: vec![Message { role: "user".into(), content: user_content }],
            temperature: invocation.runner.temperature,
        };

        let mut headers = HeaderMap::new();
        headers.insert(CONTENT_TYPE, HeaderValue::from_static("application/json"));
        headers.insert(
            "x-api-key",
            HeaderValue::from_str(&self.api_key)?,
        );
        headers.insert("anthropic-version", HeaderValue::from_static("2023-06-01"));

        let client = reqwest::Client::new();
        let resp = client
            .post("https://api.anthropic.com/v1/messages")
            .headers(headers)
            .json(&req)
            .send()
            .await?
            .error_for_status()?;

        let body: MessagesResponse = resp.json().await?;
        let text = body.content.into_iter()
            .filter(|b| b.kind == "text")
            .filter_map(|b| b.text)
            .collect::<Vec<_>>()
            .join("\n");

        // Parse the response text as a JSON object of artifact key=value pairs.
        // The runner is expected to output JSON: {"artifact_name": "value", ...}
        // File artifacts are indicated by values starting with "file://"
        let outputs = parse_agent_output(&text, &invocation.stage);

        Ok(AgentResult { outputs })
    }
}

fn build_user_message(inv: &AgentInvocation) -> String {
    let mut parts = Vec::new();
    if let Some(prompt) = &inv.prompt {
        parts.push(format!("## Task\n{}", prompt));
    }
    if !inv.artifacts.is_null() && inv.artifacts != serde_json::Value::Object(Default::default()) {
        parts.push(format!("## Input Artifacts\n```json\n{}\n```", serde_json::to_string_pretty(&inv.artifacts).unwrap_or_default()));
    }
    parts.push("## Output\nRespond with a JSON object mapping output artifact names to their values. For file artifacts, write the file and provide the path prefixed with `file://`.".to_string());
    parts.join("\n\n")
}

fn parse_agent_output(text: &str, stage: &str) -> Vec<(String, String, bool)> {
    // Extract JSON block from the response
    let json_str = extract_json(text).unwrap_or(text);
    let Ok(obj) = serde_json::from_str::<serde_json::Map<String, serde_json::Value>>(json_str) else {
        return vec![];
    };
    obj.into_iter().map(|(k, v)| {
        let val = match &v {
            serde_json::Value::String(s) => s.clone(),
            other => other.to_string(),
        };
        let is_file = val.starts_with("file://");
        let clean_val = if is_file { val[7..].to_string() } else { val };
        (k, clean_val, is_file)
    }).collect()
}

fn extract_json(text: &str) -> Option<&str> {
    let start = text.find("```json\n").map(|i| i + 8)
        .or_else(|| text.find('{'))?;
    let end = text[start..].rfind('}').map(|i| start + i + 1)?;
    Some(&text[start..end])
}

// ── Helpers ───────────────────────────────────────────────────────────────

pub fn load_items(entry: &Path) -> anyhow::Result<Vec<TlItem>> {
    let mut items = Vec::new();
    let mut seen  = HashSet::new();
    load_recursive(entry, &mut items, &mut seen)?;
    Ok(items)
}

fn load_recursive(path: &Path, items: &mut Vec<TlItem>, seen: &mut HashSet<PathBuf>) -> anyhow::Result<()> {
    let abs = path.canonicalize()
        .map_err(|_| anyhow::anyhow!("file not found: {}", path.display()))?;
    if !seen.insert(abs.clone()) {
        return Ok(());
    }
    let parsed = crate::parser::parse_file(path)?;
    let base   = abs.parent().unwrap_or(Path::new("."));
    for item in parsed {
        if let TlItem::Import(rel) = &item {
            let imported = base.join(rel);
            load_recursive(&imported, items, seen)?;
        } else {
            items.push(item);
        }
    }
    Ok(())
}

// ── Commands ──────────────────────────────────────────────────────────────

fn cmd_validate(file: &Path) -> anyhow::Result<()> {
    let items  = load_items(file)?;
    let result = validator::validate(&items);
    for w in &result.warnings {
        eprintln!("warning: {}", w.0);
    }
    if result.errors.is_empty() {
        println!("ok");
        Ok(())
    } else {
        for e in &result.errors {
            eprintln!("error: {}", e);
        }
        anyhow::bail!("{} validation error(s)", result.errors.len())
    }
}

fn cmd_inspect(file: &Path) -> anyhow::Result<()> {
    let items = load_items(file)?;
    for item in &items {
        if let TlItem::Pipeline(p) = item {
            println!("pipeline: {}", p.name);
            println!("  start: {}", p.start);
            println!("  routes:");
            for route in &p.routes {
                let src = match &route.source {
                    crate::ast::RouteSource::Stage(s) => s.clone(),
                    crate::ast::RouteSource::FanIn(s) => format!("{}[*]", s),
                    crate::ast::RouteSource::Predicate { stage, artifact, op, value } => {
                        let op_str = match op {
                            crate::ast::CompareOp::Eq => "==",
                            crate::ast::CompareOp::Ne => "!=",
                        };
                        format!("{}.{} {} \"{}\"", stage, artifact, op_str, value)
                    }
                };
                let tgt = if let Some(spec) = &route.target.parallel_spec {
                    let n = spec.limit.map(|l| l.to_string()).unwrap_or_default();
                    format!("{}[*{}]", route.target.stage, n)
                } else {
                    route.target.stage.clone()
                };
                let par = if route.parallel { " parallel" } else { "" };
                println!("    {} -> {}{}", src, tgt, par);
            }
            println!();
        }
    }
    for item in &items {
        if let TlItem::Stage(s) = item {
            let runner_name = &s.agent;
            println!("stage {} (runner: {})", s.name, runner_name);
            for a in &s.outputs {
                println!("  out: {} as {:?}", a.name, a.kind);
            }
        }
    }
    Ok(())
}

fn cmd_runs() -> anyhow::Result<()> {
    let runs = list_runs()?;
    if runs.is_empty() {
        println!("no runs found");
        return Ok(());
    }
    println!("{:<20} {:<20} {:<12} {}", "RUN ID", "PIPELINE", "STATUS", "STARTED");
    for run in runs {
        let status = match &run.status {
            RunStatus::Running => "running".to_string(),
            RunStatus::AwaitingResume { stage } => format!("waiting:{}", stage),
            RunStatus::ParallelAwait { stage, remaining } => format!("parallel:{}/{}", stage, remaining),
            RunStatus::Done => "done".to_string(),
            RunStatus::Failed(e) => format!("failed: {}", e),
        };
        println!("{:<20} {:<20} {:<12} {}", run.run_id, run.pipeline, status, run.started.format("%Y-%m-%d %H:%M"));
    }
    Ok(())
}

fn cmd_status(run_id: &str) -> anyhow::Result<()> {
    let state = RunState::load(run_id)?;
    println!("run:      {}", state.run_id);
    println!("pipeline: {}", state.pipeline);
    println!("file:     {}", state.tl_file.display());
    println!("status:   {:?}", state.status);
    println!("history:  {}", state.history.join(" → "));
    println!("updated:  {}", state.updated);
    println!("\nartifacts:");
    for key in state.artifacts.file_keys() {
        println!("  {} (file) = {}", key, state.artifacts.get_file(key).unwrap().display());
    }
    for key in state.artifacts.ref_keys() {
        println!("  {} (ref)  = {}", key, state.artifacts.get_ref(key).unwrap_or(""));
    }
    Ok(())
}

async fn cmd_run(file: &Path, pipeline_name: Option<&str>, driver_name: &str) -> anyhow::Result<()> {
    use crate::runtime::{state::RunState, Runtime};
    use uuid::Uuid;

    let items = load_items(file)?;
    let result = validator::validate(&items);
    if !result.errors.is_empty() {
        for e in &result.errors { eprintln!("error: {}", e); }
        anyhow::bail!("validation failed");
    }

    // Determine which pipeline to run
    let pipeline = match pipeline_name {
        Some(n) => n.to_string(),
        None => {
            let pipelines: Vec<_> = items.iter().filter_map(|i| {
                if let TlItem::Pipeline(p) = i { Some(p.name.clone()) } else { None }
            }).collect();
            match pipelines.len() {
                0 => anyhow::bail!("no pipeline defined in {}", file.display()),
                1 => pipelines[0].clone(),
                _ => anyhow::bail!("multiple pipelines defined — use --pipeline <name>"),
            }
        }
    };

    let run_id = Uuid::new_v4().to_string()[..8].to_string();
    let state  = RunState::new(run_id.clone(), pipeline.clone(), file.canonicalize()?);

    crate::events::ThrulineEvent::PipelineStart {
        run_id: run_id.clone(),
        ts: chrono::Utc::now(),
        pipeline: pipeline.clone(),
        inputs: serde_json::Value::Null,
    }.emit();

    let mut runtime = Runtime::new(state, items);

    match driver_name {
        "stdio" => {
            let driver = crate::driver::stdio::StdioDriver;
            runtime.advance(&driver).await?;
        }
        "api" => {
            let driver = crate::driver::api::ApiDriver::from_env()?;
            runtime.advance(&driver).await?;
        }
        _ => anyhow::bail!("unknown driver '{}' — use stdio or api", driver_name),
    }

    Ok(())
}

fn cmd_resume(run_id: &str, stage_name: &str, artifact_args: &[String]) -> anyhow::Result<()> {
    use crate::runtime::{state::RunState, Runtime};

    let state = RunState::load(run_id)?;
    let file  = state.tl_file.clone();
    let items = load_items(&file)?;
    let mut runtime = Runtime::new(state, items);

    let outputs: Vec<(String, String, bool)> = artifact_args.iter().map(|arg| {
        let (k, v) = arg.split_once('=')
            .ok_or_else(|| anyhow::anyhow!("artifact arg must be key=value, got '{}'", arg))
            .unwrap();
        let is_file = v.starts_with("file://");
        let val = if is_file { v[7..].to_string() } else { v.to_string() };
        (k.to_string(), val, is_file)
    }).collect();

    runtime.resume_stage(stage_name, outputs)?;

    // If pipeline is still running, advance to next stage
    use crate::runtime::state::RunStatus;
    if matches!(runtime.state.status, RunStatus::AwaitingResume { .. }) {
        let rt = tokio::runtime::Handle::current();
        rt.block_on(async {
            let driver = crate::driver::stdio::StdioDriver;
            runtime.advance(&driver).await
        })?;
    }

    Ok(())
}

# Thruline Skill

Runs a `.tl` pipeline file using the `thruline` binary as the orchestrator.
The skill drives agent invocations through this harness and renders live status.

## Usage

Invoke with:
```
/thruline <path-to-file.tl> [--pipeline <name>]
```

Or call from another agent context with the file path as the argument.

## How It Works

1. Runs `thruline run <file.tl> --driver stdio` as a subprocess
2. Reads NDJSON events from stdout line by line
3. On `stage_invoke` — spawns the named agent using this harness's Agent tool
4. Calls `thruline resume <run-id> --stage <name> --artifact key=value [...]` with agent output
5. Repeats until `pipeline_done` or `pipeline_error`
6. Renders live pipeline status after each event

## Display Format

After each event, output an updated status block:

```
pipeline: <name>  [run: <id>]

  ● <stage>  (<runner> / <model>)   ✓ complete
  │  <key>: <value>, ...
  ⟳ <stage>  (<runner> / <model>)   running
  │
  ○ <stage>  (<runner> / <model>)   pending
```

Legend: ● complete, ⟳ running, ○ pending

## Agent Output Protocol

When invoking a stage's agent, instruct it to respond with a JSON object:

```json
{
  "artifact_name": "value",
  "file_artifact": "file:///absolute/path/to/output.md"
}
```

File artifacts: write the file, then return `file:///path/to/file` as the value.
Ref artifacts: return the value directly as a string.

## Error Handling

- If `thruline` binary is not found: tell the user to install it (`cargo install --path .` from the thruline repo)
- If a stage agent fails: call `thruline resume <run-id> --stage <name> --artifact error=<message>` to surface the error, then halt

## Prerequisites

- `thruline` binary must be in PATH
- For stdio driver (default): no API key needed — this skill drives agents
- For api driver: `ANTHROPIC_API_KEY` must be set
